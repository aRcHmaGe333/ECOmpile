# IPConfig Protocol: Disclosure Date Continuity

Protocol ID: IPCONFIG-PROT-DATE-CONTINUITY-2026-03-04
Status: Active
Effective date: 2026-03-04

## Protocol Statement

The original disclosure date(s) in prior records still stand, even when license or IPClaim linkage metadata is updated.

## Canonical Continuity Line

The original disclosure date(s) in prior records still stand: 2025-11-08 (`README.md`, initial add in GitHub history).

## Date Source

1. Git history (`--follow --diff-filter=A`) for `README.md` -> 2025-11-08.

## Rule Application

1. License update date is additive record metadata.
2. IPClaim linkage date is additive record metadata.
3. Neither update resets or replaces original disclosure date(s).
4. Outbound provenance references must include both:
- original disclosure date(s),
- current IPClaim linkage date(s).
